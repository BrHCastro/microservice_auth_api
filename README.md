# API REST - MicroServiço de Autenticação 🔐

## 🕺 CRUD de Usuários:

- GET /users
- GET /users/:uuid
- POST /users
- PUT /users/:uuid
- DELETE /users/:uuid

## 🔓 Autenticação

- POST /token
- POST /token/validate
